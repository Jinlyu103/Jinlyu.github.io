    1. 盛最多水的容器 [中等]

      题目描述：

      给你 n 个非负整数 a1，a2，…，an，每个数代表坐标中的一个点 (i, ai) 。在坐标内画 n 条垂直线，垂直线 i 的两个端点分别为 (i, ai) 和 (i, 0)。找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。

      说明：你不能倾斜容器，且 n 的值至少为 2。

      图中垂直线代表输入数组 [1,8,6,2,5,4,8,3,7]。在此情况下，容器能够容纳水（表示为蓝色部分）的最大值为 49.

      示例：

      做这道题之前需要明确的一点是，盛水的容量相当于两根柱子之间形成的面积，而这个面积取决于较短的那根柱子的高度和柱子之间的距离。具体思路是：

      设置两个指针，i指向头，j指向尾，计算两个柱子之间形成的面积，最大面积为该面积与之前求得的最大面积之间的最大值。每次移动较短的那根柱子，循环条件为i < j。

    42. 接雨水 [**困难**]

    题目描述：

    给定 *n* 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

    ![rainwatertrap](./LeetCode题解-接雨水/rainwatertrap.png)

    上面是由数组 [0,1,0,2,1,0,1,3,2,1,2,1] 表示的高度图，在这种情况下，可以接 6 个单位的雨水（蓝色部分表示雨水）。 感谢 Marcos 贡献此图。

    示例：

    

    首先，能接雨水，那么数组的长度至少为3；

    两根柱子能够承接雨水，就要求这三个柱子形成的空间为凹的状态。定义两个指针分别为，左指针left = 0，右指针right = n-1，从左往右遍历到的最大高度记为letf_max，从右往左遍历到的最大高度记为right_max。

    在一次循环中，如果左边的高度小于右边的高度，更新left：比较height[left]和left_max的大小，如果大于或者等于，更新left_max值；否则，说明当前的柱子比之前的某根柱子矮，将left_max-height[left]的值累加到ans中（这里隐含着右边的柱子高于left_max)。最后 left += 1；

    否则，更新right：比较height[right] 和 right_max的大小，如果大于或者等于，更新right_max的值；否则，说明当前的柱子比右边的某根柱子矮，将right_max - height[right]累加到ans中（同样，这里隐含左边柱子的高度大于等于right_max)。
