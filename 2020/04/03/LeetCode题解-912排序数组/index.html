    1. 排序题

      给定一个整数数组nums，请将该数组升序排序。

      这题意思是实现一个排序算法，没有什么特殊要求。因此我实现了一个快速排序算法和一个堆排序算法。

      （一）快速排序

      1）设一个分界值key（一般取nums[left]），然后将nums按照key划分为两部分；

      2）然后对左右两部分进行独立的快速排序

      3）递归地进行，当左右排完序，整个数组也就排序完毕了

      对（nums, left, right) 进行一趟快速排序的思想是：先判断left是否大于或者等于right，如果是说明排序不用结束，直接返回，否则进行以下步骤：

      设两个指针i, j,令i = left, j = right， key = nums[left]，当i < j 时进行以下循环：

      从j 开始向前搜索(j–)，直到找到第一个小于key的值 nums[j] < key（i<j），然后交换nums[i] 和nums[j] 的值；

      从i开始向前搜索（i++），直到找到第一个大于key的值nums[i] < key (i<j)，然后交换nums[i] 和nums[j] 的值

      循环结束后，将key的值赋给nums[i], 然后分别对key左右两边的数进行递归快速排序：

      qsort(nums, left, i-1)

      qsort(nums, i+1, right)
