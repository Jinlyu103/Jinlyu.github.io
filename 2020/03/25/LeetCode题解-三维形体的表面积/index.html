      每个值 v = grid[i] [j] 表示 v 个正方体叠放在对应单元格 (i, j) 上。

      从每个单元格出发，考虑其内的立方体数量记为level（高度为1的立方体数量即为高度），其中每个立方体的表面积为四个面的面积驾驶上下两个面的面积，也就是level*4+2；

      然后需要减去前边和左边与其他单元格重合的表面积，也就是：

      min(level, grid [i-1] [j]) *2 (i >0 )

      min(level, grid [i] [j-1]) *2 (j >0 )
