    搭建个人博客

    突然想建一个私人博客，在网上找了很多资料。刚搭建完，来总结一下

    系统：

    win10

    参考：

    5分钟搭建免费个人博客

    准备：

    GitHub账号

    ​ 在GitHub上新建一个仓库，仓库名和账号名保持一致（username）

    ​ 安装Git（下一步也要安装Git，如果没有事先安装可以参考下一步的来）

    ​ 安装hexo

    配置：
    创建博客：

    ​ 在本地终端（cmd）输入：$ hexo init username.github.io

    ​ 这样就创建出了一个名为 username.github.io 的文件夹

    更改配置：

    ​ 主题安装，找一个自己喜欢的主题，然后将其 Git 到本地themes/next目录下：

    $ cd username.github.io
$ git clone https://github.com/miccall/hexo-theme-Mic_Theme.git themes/next

    ​ 基础配置：

    ​ 打开文件位置 username.github.io/_config.yml 修改键值对：

    ​ title:blog-name //博客名字

    ​ author：author-name //作者名字

    ​ language：zh-Hans //语言 中文

    ​ theme：next //刚安装的主题名称

    ​ deploy:

    ​ type: git

    ​ repo: https://github.com/username/username.github.io.git //刚创建的GitHub仓库

    所有配置好了，可以测试一下：$ hexo s

    (因为本地仓库目录下的 source/_posts/目录下有一个hello Word.md文件作为测试博客)

    测试服务启动，可以在浏览器中输入：https://localhost:4000 开始访问

    安装hexo-deployer-git自动部署发布工具：

    $ npm install hexo-deployer-git --save

    发布：

    测试成功后，将生成的静态网页文件发布至GitHub pages中：

    清除缓存命令：$ hexo clean

    生成命令并发布： $ hexo g && hexo d

    发布完成后，可以在浏览器中输入：http://username.github.io 开始访问

    最后：

    在安装的过程中大致按照上面的步骤来就OK，不过更换主题的配置有点麻烦：

    参考：如何更换博客主题

    按照这个来换主题可能会有些问题，在配置完成后需要渲染 ：

    npm install --save hexo-render-pug

    然后再重新发布到GitHub上，就OK了
